# @aitos/sense

AITOS Sense Capability Package - Motion, Environment, and Device sensing atoms.

## Installation

```bash
npm install @aitos/sense
```

## Usage

```typescript
import { allAtoms } from '@aitos/sense';

allAtoms.forEach(atom => runtime.register(atom));
```

## Atoms

- Motion sensing atoms
- Environment sensing atoms
- Device sensing atoms
