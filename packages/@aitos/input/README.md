# @aitos/input

AITOS Input Capability Package - Keyboard, Pointer, and Voice input atoms.

## Installation

```bash
npm install @aitos/input
```

## Usage

```typescript
import { allAtoms } from '@aitos/input';

allAtoms.forEach(atom => runtime.register(atom));
```

## Atoms

- Keyboard input atoms
- Pointer input atoms
- Voice input atoms
