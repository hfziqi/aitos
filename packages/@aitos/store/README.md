# @aitos/store

AITOS Store Capability Package - Local and Remote storage atoms.

## Installation

```bash
npm install @aitos/store
```

## Usage

```typescript
import { allAtoms } from '@aitos/store';

allAtoms.forEach(atom => runtime.register(atom));
```

## Atoms

- Local storage atoms
- Remote storage atoms
- Graph storage atoms
