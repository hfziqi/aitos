# @aitos/output

AITOS Output Capability Package - Visual, Audio, and Haptic output atoms.

## Installation

```bash
npm install @aitos/output
```

## Usage

```typescript
import { allAtoms } from '@aitos/output';

allAtoms.forEach(atom => runtime.register(atom));
```

## Atoms

- Visual output atoms
- Audio output atoms
- Haptic output atoms
